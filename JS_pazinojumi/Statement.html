Datorprogramma ir saraksts ar "instrukciju", lai tiktu "izpildīts" ar datoru.
Programmēšanas valodā šīs programmēšanas instrukcijas sauc par paziņojumiem .
JavaScript programma ir saraksts ar programmēšanas pārskatu .

HTML valodā JavaScript programmas izpilda tīmekļa pārlūkprogramma.

JavaScript paziņojumi sastāv no: Vērtības, operatori, izteiksmes, atslēgvārdi un komentāri.

JavaScript paziņojumi bieži sākas ar atslēgvārdu, lai identificētu veicamo JavaScript darbību.

JavaScript atslēgvārdi ir rezervēti vārdi. Rezervētos vārdus nevar izmantot kā mainīgo nosaukumus.
