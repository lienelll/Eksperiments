JavaScript sintakse ir noteikumu kopums, kā tiek veidotas JavaScript programmas

Programmēšanas valodā datu vērtības glabāšanai tiek izmantoti mainīgie .

JavaScript vērtību aprēķināšanai izmanto aritmētiskos operatorus ( + - * /)

JavaScript izmanto piešķiršanas operatoru ( =), lai piešķirtu vērtības mainīgajiem

Izteiksme ir vērtību, mainīgo un operatoru kombinācija, kas aprēķina vērtību.
Aprēķinu sauc par novērtējumu.

Vērtības var būt dažāda veida, piemēram, skaitļi un virknes.
